# Rivals Scripts

Roblox Rivals executor scripts with UI, skin changer, and exploits.

## Files

| File | Size | Description |
|------|------|-------------|
| `newrivals2.lua` | 846 KB | Main script — full UI, skin changer (Skins/Wraps/Charms/Finishers/Misc tabs), movement, aimbot, esp |
| `newrivals.lua` | 724 KB | Previous version |
| `loader.lua` | ~ | Simple loader — `loadstring(game:HttpGet("..."))` |

## Usage

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/zaisaawe21/rivals-scripts/main/loader.lua"))()
```

Or load a specific script directly:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/zaisaawe21/rivals-scripts/main/newrivals2.lua"))()
```
